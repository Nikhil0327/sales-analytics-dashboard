# Sales Analytics Dashboard (Power BI)

## Project Overview

Developed an interactive Sales Analytics Dashboard using Power BI to analyze sales performance, profit trends, order volume, and regional performance. The dashboard enables users to explore sales data through interactive filters and visualizations, helping identify key business trends and insights.

---

## Dataset

This project uses the **Sample Superstore Dataset**, which contains information related to:

* Orders
* Sales
* Profit
* Product Categories
* Regions
* Customer Segments
* Order Dates

The dataset file is included in this repository for reference and reproducibility.

---

## Features

* KPI Cards for Total Sales, Total Profit, and Total Orders
* Category-wise Profit Analysis using Donut Chart
* Region-wise Sales Comparison using Bar Chart
* Monthly Sales Trend Analysis using Line Chart
* Interactive Filters (Region, Category, Date)
* Business Insights Section
* Clean and Professional Dashboard Layout

---

## Tools & Technologies Used

* Power BI Desktop
* DAX (Data Analysis Expressions)
* Data Modeling
* Data Visualization
* Business Intelligence Techniques

---

## Key Insights

* West region generated the highest overall sales.
* Technology category contributed the highest profit share.
* Sales performance showed strong growth during 2017.
* South region recorded comparatively lower sales volume.
